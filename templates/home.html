API Documentation

Create Device

Endpoint: /api/devices/
Method: POST
Description: Create a new device.

Get Device

Endpoint: /api/devices/<str:uid>/
Method: GET
Description: Retrieve information about a specific device.

Reading View

Endpoint: /api/devices/<str:uid>/readings/<str:parameter>/
Method: GET
Description: Get readings (temperature or humidity) for a specific device in a given period.

Device Graph View

Endpoint: /devices-graph/<str:uid>/
Method: GET
Description: View a graph of temperature and humidity vs time for a specific device.
